# bioSite (Module 4)

This package contains the updated site matching the Module 4 PDF.

## Structure
- index.html, about.html, hobbies.html, Contact.html
- styles.css
- assets/images/{books.png, garden.png, meal.png, collage.png, portrait.svg}

## How to use
1. Unzip into your local repo folder (e.g., C:\csd\csd-340\bioSite\).
2. Ensure the `assets/images/` paths match the HTML.
3. Git push:
   ```powershell
   git add -A
   git commit -m "Restore bioSite (Module 4)"
   git push origin main
   ```
